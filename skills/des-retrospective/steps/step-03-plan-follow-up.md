# Step 03 - Plan Follow-Up

Create actionable backlog items.

For each follow-up include:

- Title.
- Owner or responsible role.
- Severity or priority.
- Source evidence.
- Next workflow: phase skill, `des-brainstorm-change`, `des-implementation-planning`, `des-dev-story`, or `des-project-evaluation`.

Group follow-up by artifact updates, implementation debt, tests/evidence, operations, and process improvements.
