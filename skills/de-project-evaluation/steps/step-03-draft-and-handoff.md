## Hướng dẫn

### 1. Nạp template
Đọc: 22-project-evaluation-template.md

### 2. Soạn thảo artifact
Kết quả thực tế, so sánh vs cam kết, go/no-go decision với rationale.

### 3. Ghi file và cập nhật trạng thái
Lưu vào 22-project-evaluation.md. Update workflow status. Đánh dấu project phase COMPLETED.

- **[R] Soạn lại**: Quay lại step-01.
