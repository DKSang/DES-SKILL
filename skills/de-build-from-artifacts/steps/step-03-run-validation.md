# Step 03 - Run Validation

Collect fresh evidence.

1. Run planned tests, validators, linters, or build commands.
2. Add focused checks for modified behavior when the plan missed an obvious validation.
3. Capture command, result, and important failure details.
4. Fix failures that are inside planned scope.
5. Mark unresolved failures as blockers.

Do not reuse stale command output. Fresh evidence is required before a completion claim.
