# Step 03 - Run Validation

Collect fresh evidence.

1. Run the RED/GREEN/REFACTOR task tests first for behavior changes.
2. Run planned tests, validators, linters, or build commands.
3. Add focused checks for modified behavior when the plan missed an obvious validation.
4. Capture command, result, and important failure details.
5. Fix failures that are inside planned scope.
6. Mark unresolved failures as blockers.
7. Do not mark complete when any required evidence is missing, stale, skipped without accepted reason, or failing.

Do not reuse stale command output. Fresh evidence is required before a completion claim.
