# Step 01 - Discover Inputs

Collect the implementation source of truth.

1. Read the approved DES artifacts and `change-brief.md` when present.
2. Read prior `implementation-log.md`, `review-report.md`, and `verification-report.md` if this is follow-up work.
3. Inspect repo structure only enough to identify likely files, tests, and commands.
4. Build a source artifact map with artifact path, relevant requirement, and expected implementation impact.
5. List unresolved questions before drafting tasks.
