# Checklist — des-artifact-quiz

## Source Context

- [ ] Source artifact or learning notes are identified.
- [ ] DES phase is identified.
- [ ] Related concept map section is used.
- [ ] User learning goal is clear or defaulted safely.

## Quiz Coverage

- [ ] Concept-check questions are included.
- [ ] Artifact-understanding questions are included.
- [ ] Applied scenario questions are included.
- [ ] Trade-off questions are included.
- [ ] Common-mistake questions are included.
- [ ] Questions cover both business and technical reasoning where relevant.

## Question Quality

- [ ] Questions test understanding, not memorization only.
- [ ] Questions are clear and answerable.
- [ ] Questions are tied to the artifact or phase.
- [ ] Difficulty levels are mixed.
- [ ] No unsupported concepts are introduced.

## Answer Quality

- [ ] Answer key is included.
- [ ] Each answer includes explanation.
- [ ] Explanations teach the reasoning.
- [ ] Scoring guide is included.
- [ ] Next action is recommended.

## Guardrails

- [ ] Source artifact is not modified.
- [ ] No long source passages are reproduced.
- [ ] No invented project context is used.
- [ ] The quiz does not claim mastery automatically.
