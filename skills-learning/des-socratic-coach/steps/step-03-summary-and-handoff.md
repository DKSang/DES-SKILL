# Step 03 — Summary and Handoff

## Goal

Summarize the Socratic session and recommend the next learning action.

---

## Required Reading

Read:

```text
skills-learning/des-socratic-coach/checklist.md
```

---

## Session Summary Must Include

1. coaching topic;
2. target concept;
3. user's starting understanding;
4. improved understanding;
5. remaining misconception;
6. final mental model;
7. artifact connection;
8. recommended next action.

---

## Status Values

Use one of:

```text
Draft
Complete
Complete with Misconceptions
Blocked
Waiting for User Answer
```

Use `Waiting for User Answer` in interactive mode after asking a question.

Use `Complete with Misconceptions` when the user improved but still has weak understanding.

---

## Handoff Rules

Recommend one next action:

### Run Quiz

Use when the user should verify understanding.

### Run Gap Teacher

Use when misconceptions remain.

### Explain Artifact Again

Use when fundamentals remain weak.

### Improve Artifact

Use when the coaching revealed artifact problems.

### Continue Lifecycle

Use only when the final synthesis is strong enough.

---

## Final Output

End with:

```md
## Checklist Result

- Checklist status:
- Coaching status:
- Remaining misconception:
- Recommended next action:
- Next recommended skill:
```
