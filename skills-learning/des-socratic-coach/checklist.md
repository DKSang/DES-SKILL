# Checklist — des-socratic-coach

## Context

- [ ] Coaching topic is identified.
- [ ] DES phase is identified where relevant.
- [ ] Target concept is identified.
- [ ] Coaching mode is selected.
- [ ] User confusion or learning goal is captured.

## Question Design

- [ ] Questions progress from simple to applied.
- [ ] Questions are tied to the artifact or concept.
- [ ] Questions test reasoning, not memorization.
- [ ] Hints are included.
- [ ] Misconception checks are included.
- [ ] Trade-offs are included where relevant.

## Interactive Coaching

- [ ] Only one main question is asked at a time.
- [ ] The agent waits for the user's answer.
- [ ] The agent does not reveal the final answer too early.
- [ ] The agent corrects misconceptions gently.
- [ ] The agent summarizes the mental model.

## Output

- [ ] Coaching session is saved or summarized.
- [ ] Final synthesis is included.
- [ ] Artifact connection is included.
- [ ] Next recommended skill is included.

## Guardrails

- [ ] No unsupported concepts are introduced.
- [ ] No mastery claim is made without evidence.
- [ ] Source artifact is not modified unless explicitly requested.
