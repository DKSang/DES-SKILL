# Step 03 — Summary and Handoff

## Goal

Validate the learning review and recommend the next action.

---

## Required Reading

Read:

```text
skills-learning/des-learning-review/checklist.md
```

---

## Validation Procedure

Check that the review includes:

1. overall progress;
2. module progress;
3. phase readiness;
4. strengths;
5. weak concepts;
6. open gaps;
7. blockers;
8. review plan;
9. next learning action;
10. next DES lifecycle action where safe.

---

## Status Values

Use one of:

```text
Draft
Complete
Complete with Unknowns
Blocked
Needs Status Update
```

Use `Needs Status Update` when learning status is missing or stale.

Use `Complete with Unknowns` when the review is useful but some learning evidence is missing.

---

## Handoff Rules

Recommend one next action:

### Update Learning Status

Use when status is missing or stale.

### Explain Missing Artifact

Use when an artifact exists but no learning notes exist.

### Generate Quiz

Use when explanation exists but quiz is missing.

### Diagnose Gap

Use when answers exist but gap report is missing.

### Coach Concept

Use when a conceptual gap remains.

### Continue Lifecycle

Use only when readiness is sufficient.

---

## Final Output

End with:

```md
## Checklist Result

- Checklist status:
- Learning review status:
- Overall readiness:
- Main blocker:
- Recommended next action:
- Next recommended skill:
```
