# Step 03 — Checklist and Handoff

## Goal

Validate the learning path and recommend the next action.

---

## Required Reading

Read:

```text
skills-learning/des-learning-path/checklist.md
```

---

## Validation Procedure

Check that the learning path includes:

1. learner goal;
2. module structure;
3. all 22 phases;
4. concept mapping;
5. artifact exercises;
6. quiz checkpoints;
7. weak areas if evidence exists;
8. recommended study order;
9. next recommended skill.

---

## Status Values

Use one of:

```text
Draft
Complete
Complete with Gaps
Blocked
Generic Plan
```

Use `Generic Plan` when no project artifacts exist and the path is not personalized.

Use `Complete with Gaps` when some artifact statuses are unknown.

---

## Handoff Rules

Recommend one next action:

### Start With Phase 01

Use when no artifacts exist.

### Explain Existing Artifact

Use when an artifact exists but learning notes are missing.

### Run Artifact Quiz

Use when learning notes exist but understanding has not been checked.

### Continue DES Lifecycle

Use when the current module is complete.

### Update Workflow Status

Use when learning status tracking is enabled.

---

## Final Output

End with:

```md
## Checklist Result

- Learning path status:
- Personalization level:

- Recommended next action:

```
