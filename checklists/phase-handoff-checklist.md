# Phase Handoff Checklist

Use this checklist to verify that a phase handoff is safe for the next phase.

## Metadata

- [ ] From phase is identified.
- [ ] To phase is identified.
- [ ] From skill is identified.
- [ ] Next skill is identified.
- [ ] Phase artifact is referenced.
- [ ] Evidence pack is referenced.
- [ ] Done Gate is referenced.
- [ ] Handoff status is selected.

## Final Decisions

- [ ] Final decisions are listed.
- [ ] Decision evidence is referenced.
- [ ] Draft decisions are clearly marked.
- [ ] Open decisions are not presented as approved.
- [ ] Reversibility is documented when relevant.

## Approved Inputs

- [ ] Next phase inputs are listed.
- [ ] Input source artifact or evidence is referenced.
- [ ] Approved inputs are separated from assumptions.
- [ ] Input limitations are documented.

## Constraints

- [ ] Constraints for next phase are listed.
- [ ] Reason for each constraint is documented.
- [ ] Constraint source is referenced.
- [ ] Hard constraints are clear.

## Risks

- [ ] Risks carried forward are listed.
- [ ] Severity is assigned.
- [ ] Impact on next phase is documented.
- [ ] Required handling is documented.
- [ ] Accepted risks are not hidden.

## Open Questions

- [ ] Open questions are listed.
- [ ] Owner is listed where possible.
- [ ] Needed-by phase is listed.
- [ ] Impact is documented.

## Do-Not-Assume List

- [ ] Do-not-assume items are listed.
- [ ] Items are specific and actionable.
- [ ] The next phase knows what requires evidence or user decision.

## Evidence References

- [ ] Evidence references are listed.
- [ ] Evidence relevance to next phase is explained.
- [ ] Missing evidence is not hidden.

## Next Phase Instructions

- [ ] Recommended next skill is listed.
- [ ] Starting instructions are provided.
- [ ] Required files to read are listed.
- [ ] Handoff decision is selected.

## Completion

- [ ] Handoff is Ready, Ready with Risks, Not Ready, or Blocked.
- [ ] If Ready with Risks, conditions are listed.
- [ ] If Not Ready or Blocked, required actions are listed.
