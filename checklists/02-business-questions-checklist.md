# Checklist - Business Questions

## Required Context

- [ ] Phase 1 Business Discovery artifact exists or Draft continuation is explicitly accepted.
- [ ] Primary consumers are identified.
- [ ] Target decisions or use cases are identified.
- [ ] Project scope and non-scope are considered.
- [ ] Expected value is considered.

## Question Quality

- [ ] Each P1 question maps to a consumer.
- [ ] Each P1 question maps to a decision or use case.
- [ ] Each P1 question has clear business value.
- [ ] Each P1 question is written in business language.
- [ ] Each P1 question is plausibly answerable with data.
- [ ] Each P1 question has an expected answer format.
- [ ] Each P1 question has a priority.
- [ ] Question status is marked as Draft, Approved, Deferred, or Out of Scope.

## Guardrails

- [ ] The catalog does not define final KPI formulas.
- [ ] The catalog does not design Gold tables.
- [ ] The catalog does not choose final data sources.
- [ ] The catalog does not design dashboards, APIs, semantic models, ML models, or pipelines.
- [ ] Candidate metrics are clearly marked as candidates.
- [ ] Source hints are clearly marked as hints.
- [ ] Grain hints are clearly marked as hints.
- [ ] Assumptions are explicitly marked.
- [ ] Scope conflicts are documented.

## HALT Validation

- [ ] Missing Phase 1 context HALT resolved or artifact remains Draft.
- [ ] Question priority HALT resolved or artifact remains Draft.
- [ ] Scope conflict HALT resolved or artifact remains Draft.
- [ ] Answerability risk HALT resolved or artifact remains Draft.
- [ ] Consumer mapping HALT resolved or artifact remains Draft.
- [ ] Decision mapping HALT resolved or artifact remains Draft.

## Result

Choose one:

- [ ] Passed
- [ ] Passed with risks
- [ ] Blocked
