# Definition of Done Checklist

Use this checklist before marking implementation ready-for-review, complete, released, or delivery-supported.

## Tasks Complete

- [ ] All planned tasks and subtasks are checked in `implementation-story.md`.
- [ ] Any skipped task has a reason and owner.
- [ ] No unplanned scope is hidden as completed work.
- [ ] Status is Ready for Review, Blocked, or Complete.

## Acceptance Criteria Satisfied

- [ ] Every acceptance criterion is satisfied or explicitly blocked.
- [ ] Each criterion has linked evidence.
- [ ] Data contract, DQ, lineage, governance, and security acceptance criteria are included when relevant.
- [ ] Non-functional criteria such as SLA, cost, and performance are checked when relevant.

## Fresh Evidence

- [ ] Verification commands were run in the current workspace/session.
- [ ] Command names and outcomes are recorded.
- [ ] Failures and skipped commands are visible.
- [ ] Manual verification evidence is reproducible.

## Changed File List

- [ ] `implementation-story.md` File List includes every new, modified, and deleted file.
- [ ] File changes map to tasks or review follow-ups.
- [ ] No unrelated files are included without explanation.
- [ ] Generated artifacts are named separately from source edits.

## Review Blockers

- [ ] Review findings are resolved, accepted, or still blocking.
- [ ] No high/blocker finding is hidden in notes.
- [ ] Accepted risks include owner and next action.
- [ ] Review report recommendation matches current status.

## Artifact Drift

- [ ] Any mismatch between implementation and planning artifacts is recorded.
- [ ] Drift is routed to retrospective, Correct Course, or the affected phase skill.
- [ ] Follow-up owner and severity are named.
- [ ] The next workflow action is explicit.

## Completion Decision

| Decision | Meaning |
| :--- | :--- |
| Supported | DoD passes and evidence supports completion claim |
| Partial | Some work is complete, but named risks or skipped checks remain |
| Blocked | Required task, evidence, or review gate fails |
