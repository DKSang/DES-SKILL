# Lineage And Metadata Checklist

- [ ] Dataset-level lineage covers source to serving.
- [ ] Column-level lineage exists for critical fields.
- [ ] Metric lineage traces certified metrics to source fields.
- [ ] Dataset owners, stewards, and consumers are recorded.
- [ ] Sensitive data classification is present.
- [ ] Downstream dependencies are documented.
- [ ] Catalog metadata can be maintained or generated.
