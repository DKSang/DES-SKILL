# Example Ecommerce Project

## Goal
Build a sales and customer behavior analytics platform for an ecommerce business.

## Sources
- Orders database.
- Product catalog.
- Web analytics events.
- Payment processor exports.
- Marketing campaign data.

## Gold Models
- `fact_order`.
- `fact_web_session`.
- `dim_customer`.
- `dim_product`.
- `mart_sales_daily`.

## Serving
- Executive revenue dashboard.
- Product performance dashboard.
- API for customer lifetime value.
